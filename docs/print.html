RE_DirectTrampolineHook

void RE_DirectTrampolineHook(void* start, void* cbFunc, void** oldFunc);

Input:

  • start: function pointer that will be hooked.
  • cbFunc: function that gets called instead of the original one

Output:

  • oldFunc: pointer to the original Function

Example:

Hooking MessageBoxA:

static int(*originalMsgBoxFunc)(HWND, LPCSTR, LPCSTR, UINT) = nullptr;
int MessageBoxAHook(HWND hWnd, LPCSTR lpText, LPCSTR lpCaption, UINT uType)
{
    // DO SOMETHING MAYBE NOT EVEN CALL ORIGINAL FUNCTION
    return originalMsgBoxFunc(hWnd, lpText, lpCaption, uType);
}

void ExampleMain()
{
    void* messageBoxFunc = (void*)GetProcAddr(GetModuleHandle(NULL), "MessageBoxA");
    RE_DirectTrampolineHook(messageBoxFunc, 
                            (void*)MessageBoxAHook, 
                            (void**)&originalMsgBoxFunc);
}

RE_EasyTrampolineHook

void RE_EasyTrampolineHook(void* start, void* cbFunc);

Input:

  • start: function pointer that will be hooked.
  • cbFunc: function that gets called before the original one

Info:

In x86 the function will have the return value pushed on the stack theirfore the __stdcall functions will have an additional parameter before the others

Example:

Hooking MessageBoxA:

#ifdef _WIN64
void MessageBoxAHook(HWND hWnd, LPCSTR lpText, LPCSTR lpCaption, UINT uType)
{
    // DO SOMETHING
}
#else
void MessageBoxAHook(void* r, HWND hWnd, LPCSTR lpText, LPCSTR lpCaption, UINT uType)
{
    // DO SOMETHING
}
#endif

void ExampleMain()
{
    void* messageBoxFunc = (void*)GetProcAddr(GetModuleHandle(NULL), "MessageBoxA");
    RE_EasyTrampolineHook(messageBoxFunc, (void*)MessageBoxAHook);
}

RE_CustomTrampolineHook

void RE_CustomTrampolineHook(void* start, CustomFunc cbFunc);

Input:

  • start: function pointer that will be hooked.
  • cbFunc: function that gets called instead of the original one

Info:

IMPORTANT: THIS HOOK IS NOT THREAD SAFE
this function can be used to hook anywhere inside a function as long as the code flow allows it.
invalid places are:

  • less than 5 instructions before a relative instruction (x64 only)
  • overwriting inbetween a jump destination

Example:

Hooking MessageBoxA:

CPU_STATE* MessageBoxAHook(CPU_STATE* state)
{
    // DO SOMETHING
    return state;
}
void ExampleMain()
{
    void* messageBoxFunc = (void*)GetProcAddr(GetModuleHandle(NULL), "MessageBoxA");
    RE_CustomTrampolineHook(messageBoxFunc, MessageBoxAHook);
}

{

#endif
}

RE_ReadPointerRPM

void* RE_ReadPointerRPM(void* base, const std::vector<intptr_t>& offsets);
void* RE_ReadPointerRPM(const PTR& pointer);

Input:

  • base: first address to unrefrence
  • offsets: offsets added onto the unrefrenced values
  • or pointer: combination of both base and offsets

Output:

Pointer to the last memory location
can be nullptr if the function fails

Info:

Reads the Pointer by using ReadProcessMemory to validate the memory location.
This function can't crash even with ridiculous values.
